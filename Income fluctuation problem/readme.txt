Replication files for "Capital and Labor Income Pareto Exponents across Time and Space", Section 4
by Tjeerd de Vries and Alexis Akira Toda, Review of Income and Wealth

Confirmed to run with Matlab 2021b

Main files:
	- IFP.m: output results in Section 4.3

Other files and folders:
	- PE: codes for Pareto extrapolation algorithm
	- expGrid.m: constructs exponential grid
	- GaussHermite.m: code for Gauss-Hermite quadrature
	- getC.m: code for computing consumption functions by endogenous grid point method
